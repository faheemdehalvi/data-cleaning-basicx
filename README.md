## Overview

This repository contains a simple and practical notebook-based assignment focused on:

- Inspecting raw data
- Handling missing values
- Cleaning and preparing data for analysis
- Documenting outcomes in a report

## Project Files

| File | Description |
|---|---|
| `assignment1.ipynb` | Main notebook with data cleaning steps and outputs |
| `report.pdf` | Final assignment/report document |

## Quick Start

1. Clone the repository:

	 ```bash
	 git clone https://github.com/faheemdehalvi/data-cleaning-basicx.git
	 cd data-cleaning-basicx
	 ```

2. Open the notebook in Jupyter or VS Code.

3. Run cells top-to-bottom to reproduce the cleaning workflow.

## Recommended Environment

- Python 3.10+
- Jupyter Notebook / JupyterLab
- VS Code (optional)

## Workflow Snapshot

```text
Raw Data -> Inspect -> Clean -> Validate -> Document
```
